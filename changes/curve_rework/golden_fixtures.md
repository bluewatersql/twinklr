# Golden Fixtures (Task 0.3)

## Sampling conventions
- Use `sample_uniform_grid(n)` with `n=8`.
- The grid is `[0, 1/8, 2/8, ..., 7/8]` (no `t=1.0`).
- Values are rounded to 6 decimal places for fixtures.
